JavaScript and Looping

What is looping?

Need it done again and again? Looping is a programming structure that allows you to repeat a set of code in a various ways

For Loops

To iterate/go over through code a set number of times, the for loop is what should be used.

While Loops

To execute a set of code repeatedly until condition is met to then exit the loop.

Filter

Filter loops are used with arrays to filter/keep items in a new array based on a boolean test, if the array item passes the boolean (is true), it is filtered in the array, false items are filtered out

Map

Map iterates/goes over each item in an array and processes each with some code.
