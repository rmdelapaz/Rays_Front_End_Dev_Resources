JavaScript Vocabulary

Basic terms and concepts

All languages, whether spoken, programming, or otherwise has a structure, a syntax. A way to say or do things. To communicate successfully with this syntax, we need to know and use the languages vocabulary appropriately. In english, knowing what a noun, adjective or verb is very important to help you increase your language vocabulary. The same is true for programming lanuages, knowing what key terms are such as keywords, identifiers, operators, statements help greatly in understanding.

Glossary of Programming Terms

keyword
words that are part of the language, a term you cannot use as an identifier
identifier
a developer created name for a programming resource, a variable, function and so on
variable
a container of information, could be simple where it holds a single primitive or more complex like and object or array.
data structure
a more complex variable that holds a variety of information tied to an identifier
