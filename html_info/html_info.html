What is Hyper Text Markup Language

Hyper Text Markup Language is what web developers use to identify/"markup" the content.The first version of Hyper Text Markup Language, created by Tim Berners-Lee in 1993 using Structured Generalized Markup Language. As a scientist whose intent was to share information, attention was given more to information itself rather than the aesthetics. HTML is what web developers use to define and identify the web content through the use of HTML "tags". These tags are contained in less than (<:) and greater than (>) symbols. These tags often come in pairs with a beginning tag marking the start of the content identified and at the end, a corresponding closing tag the would have a slash (/) right after the less than symbol of the closing tag.

Some mistakenly claim/think that HTML is a programming language. Um, it's in the name. It is a markup language that has no logic or algorithm. All it has is markup for the data structure.
