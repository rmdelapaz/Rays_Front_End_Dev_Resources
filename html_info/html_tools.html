What can you use to create HTML files?

An HTML file is nothing more than a simple text file. You can use a simple text editors such as notepad or textpad to create HTML files. Applications such as Adobe Dreamweaver, Microsoft Visual code, and other power tools help in other parts of the process.

What do we use to view HTML Files
