Python

What is it?

A popular programming language that can be used for a variety of applications

Why learn and use it?

It's a language that is used for web development, Artificial intelligence, data mining and other powerful applications.

If it's your first programming language, it is one of the easier C like programming language out there. What do I mean by C like? C is one of the first popular programming language made. As such, other programming languages created after made use of that popularity and built similar, C like languages, C++, Java, PHP, Javascript and here we are with Python. Similarities with some of the syntax rules will be all over the place

If have experienced Javascript or some other C like programming language, you will see a lot of similarities in the syntax. The use of curly braces and semi colons.

Most programming languages have loops and if statements.

Dowloading and installing Python

If you did start your programming adventure with javascript, you didn't have to download and install anything if you already had a modern web browser. Modern web browsers have the capability of processing and understanding javascript built in as part of its standard. You don't need anything special to create javascript, a text editor works. For Python, it's a different story. You need to download and install python and an application that makes creating python code easier.

The address for Python downloads is: https://www.python.org/downloads/

Scroll down to the appropriate link for the operating system you are using. (windows, macOS, Linux/Unix or other)

Click on appropriate link to download Python installer. (as of last check, latest version is 3.12.5)

Once it's downloaded, double click on the downloaded installer to run. Have it create appropriate paths.
