Python Comments

What are Python Comments?

Comments are text that is not treated a processed code. It's meant to serve as information

When and Why use Javascript Comments

Javascript comments provide a way for developers to provide insights or reminders about certain parts of the code they are working with within the codebase itself.

How to create Javascript Comments

Single Line Comments

Using slash slash will start a single line javascript comment and hitting enter/return will end the single line comments

Multi Line Comments

A multi-line comment begins with a slash asterish and ends with an asterisk slash.
