Python Comments

What are Python Comments?

Comments are text that is not treated a processed code. It's meant to serve as information

When and Why use Python Comments

Python comments provide a way for python developers to provide insights or reminders about certain parts of the code they are working with within the codebase itself.

How to create Javascript Comments

Single Line Comments

Python used the number sign/pound sign/octotorph/hash/sharp symbol aka shift 3 on your keyboard for a single line comment.

Multi Line Comments

A multi-line comment begins with a slash asterish and ends with an asterisk slash.
