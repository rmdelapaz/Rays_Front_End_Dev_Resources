Command Line Interface

Keyboards and useful keys

Keyboard differences

Being able to navigate and manipulate files and folders/directories is a skill you probably have mastered using the GUI and mouse. The command line interface (CLI for short) is also known as terminal (mac/linux), gitbash or powershell (windows).

Universal Keys that you may or may not have

End key and Home key. F12. FN.

Universal keys that you should have available

Some of these keys you've probably never used if ever. Curly Brace, Square Brace, Angle Brace. Parenthesis, Back Tick, Tilde, Esc.
