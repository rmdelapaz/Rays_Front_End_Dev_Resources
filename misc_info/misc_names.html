Names aka Identifiers in Front End Web Development

Identifiers in different technologies

Generic Rules

When in doubt, it is case-sensitive, cat and CAT do not refer to the same piece of information.

You use identifiers in HTML, CSS and JavaScript to uniquely identify typically elements for HTML and CSS and data items and structures for Javascript. Depending on who you work for or with, what naming conventions you end up using might vary but for the most part, most developers, whether for good or bad has adopted certain naming conventions.
